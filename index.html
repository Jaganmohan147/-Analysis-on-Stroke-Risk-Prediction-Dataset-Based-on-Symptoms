Analysis on Stroke Risk Prediction Dataset Based on Symptoms

Stroke is a serious medical condition that happens when blood flow to the brain is blocked. Early prediction of stroke can save lives. This project analyzes a stroke risk dataset using Pandas Profiling. The goal is to find patterns in symptoms that indicate high stroke risk. This helps in early detection and better prevention.

Abstract

Introduction

A stroke occurs when the brain does not get enough blood supply. It can lead to serious health problems or even death. Common stroke symptoms include high blood pressure, diabetes, smoking, and obesity. Detecting stroke risk early can help in taking preventive measures. In this project, we analyze a stroke dataset to find patterns using Pandas Profiling. This tool helps in identifying important risk factors easily.

Literature Survey

Many studies have focused on stroke risk prediction using medical data. Some researchers use machine learning models to classify stroke risk levels. Others perform statistical analysis to find common symptoms in stroke patients. Traditional methods rely on doctor evaluation, but automated analysis can improve accuracy. Studies show that Exploratory Data Analysis (EDA) helps in understanding stroke risk factors effectively.

EDA

EDA helps in analyzing patient symptoms and medical history. Using Pandas Profiling, we can see patterns in age, blood pressure, cholesterol levels, and smoking habits. Stroke patients often have high blood pressure, diabetes, or heart disease. EDA helps to find these risk patterns quickly.

Technology Used
Python – For coding and data analysis.

Pandas Profiling – To generate automatic data reports.

Jupyter Notebook – To write and run the code.

Matplotlib & Seaborn – To create graphs and visualize stroke risk factors.

Machine Learning (optional) – If needed, models like Logistic Regression or Random Forest can predict stroke risk.
